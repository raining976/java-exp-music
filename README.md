# exp04

音乐播放器客户端

## 主要功能实现

1. 获取共享歌单列表以及歌单相关信息
2. 下载歌曲和封面
3. 添加本地歌单和删除本地歌单
4. 实现了播放暂停等相关功能


## 开发期间遇到的问题以及如何解决的

1. 各个gui组件之间的通讯问题:
	基本都用根组件 MusicPlayerGUI做了父子通讯
2. 本地数据库不能创建的问题
	因为我用的是mysql,所以有部分语法不匹配,干脆直接使用了workbench可视化手动创建了数据库和表
3. 播放暂停功能:
	卡了很久,最后还是写的多线程处理的播放暂停,期间也有一些不太合理的地方,都做了修改
